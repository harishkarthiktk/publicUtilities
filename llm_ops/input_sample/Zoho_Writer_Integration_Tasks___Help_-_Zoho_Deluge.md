# Zoho Writer Integration Tasks

* [Get Documents](/deluge/help/writer/get-documents.html): Lists documents from Zoho Writer based on a condition
* [Upload Document](/deluge/help/writer/upload-document.html): Uploads a document to Zoho Writer
* [Share Document](/deluge/help/writer/share-document.html): Shares a document with your list of collaborators, based on roles
* [Get Merge Fields](/deluge/help/writer/get-merge-fields.html): Lists all the merge fields inserted in the specified document
* [Merge And Send](/deluge/help/writer/merge-and-send.html): Auto-populates merge values and emails the document to a specified email address
* [Sign document](/deluge/help/writer/sign-documents.html): Sends the document to a list of recipients for signature collection
* [Get All Fields](/deluge/help/writer/get-all-fields.html): Lists all the fields in the specified document
* [Merge and Sign](/deluge/help/writer/merge-and-sign.html): Merges the document and sends it to recipients for signature collection
* [Merge and Store V2](/deluge/help/writer/merge-and-store-v2.html): Merges and stores the document in a specific location in  Zoho WorkDrive
* [Merge and Invoke](/deluge/help/writer/merge-and-invoke.html): Merges the document and invokes a custom function to perform further operations
* [Mark or Unmark Favorite Document](/deluge/help/writer/mark-unmark-favorite-document.html): Marks or unmarks the document as favorite
* [Rename Document](/deluge/help/writer/rename-document.html): Renames the document with the given name
* [Enable or Disable Track Changes](/deluge/help/writer/enable-disable-track-changes.html): Enables or disables track changes for the specified document
* [Add or Update Description](/deluge/help/writer/add-update-description.html): Adds or updates the document description with the given description
* [Lock or Unlock Documents](/deluge/help/writer/lock-unlock-documents.html): Locks or unlocks the particular document
* [Mark Document as Ready](/deluge/help/writer/mark-document-ready.html): Marks the specific document as ready
* [Mark or Revert Final Documents](/deluge/help/writer/mark-revert-final-documents.html): Enables or disables editing in the document
* [Get Merge Templates](/deluge/help/writer/get-merge-templates.html): Lists all the merge templates created by a user
* [Get Sign Templates](/deluge/help/writer/get-sign-templates.html): Lists all the sign templates created by a user
* [Get Fillable Templates](/deluge/help/writer/get-fillable-template.html): Lists all the fillable templates created by a user
* [V2 Sign Documents](/deluge/help/writer/v2-sign-document.html): sends a document for signature and tracks its status
* [Generate Fillable Link](https://www.zoho.com/deluge/help/writer/GenerateFillableLink.html): merges your document and generates prefilled fillable links to collect data
* [Merge And Send V2](https://www.zoho.com/deluge/help/writer/MergeAndSendV2.html): merges your document and sends it via email
